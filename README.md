## Brief description
Cartogra-fun is an interactive map projection playground that lets you compare a rotatable globe with a flat map, switch between projection families, inspect distortion, trace geodesics, and explore day-night shading.

## Instructions for use
Choose a projection from the top bar, drag the globe to change the view, use Select Point to inspect distortion at a location, use Geodesic to place two points and compare the shortest path on the globe and map, and toggle Fix Projection Map, Show Projection Surface, or Day/night to explore the projection behavior.
